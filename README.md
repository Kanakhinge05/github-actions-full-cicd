# 🍴 My Recipe Hub

**My Recipe Hub** is a simple web application where users can explore, add, and manage their favorite recipes.  
It allows users to search recipes, add new ones, and view recipes added by others.

---

## 🌟 Features

- 🔍 **Search Recipes** – Find recipes using keywords.
- 📖 **View All Recipes** – Browse the complete recipe collection.
- ➕ **Add Recipe** – Users can add their own recipes.
- 👤 **User Login System** – Simple login using browser storage.
- 📊 **Recipe Statistics** – Shows total recipes and recently added recipes.
- 🍳 **Recipe Spotlight** – Displays a random recipe.

---

## 🖥️ Website Pages

| Page | Description |
|-----|-------------|
| `index.html` | Homepage showing recipe highlights |
| `search.html` | Search recipes by keyword |
| `recipes.html` | View all available recipes |
| `add-recipe.html` | Add a new recipe |
| `login.html` | User login/signup page |
| `account.html` | Manage user account |
